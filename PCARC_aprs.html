PCARC APRS

APRS or Automatic Packet Reporting System can be used to share your location or send text messages via a radio. These messages can be sent digitally or via an analogue signal. An Ametur Radio License is required to send this information but not necesseraily to recive. See the links below for mor information regarding APRS.
